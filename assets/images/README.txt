# Image Placement Guide

This directory is where all website images should be placed. Please add the following images:

## Required Logo Files
- logo.png - Main website logo (displayed in header)
- logo-white.png - White version of logo for footer

## Background Images
- hero-bg.jpg - Hero section background
- about-bg.jpg - About section background
- services-bg.jpg - Services section background

## Team Member Photos
- team-member1.jpg - CEO photo
- team-member2.jpg - CSO photo
- team-member3.jpg - COO photo

## Other Images
- about-preview.jpg - Image for about section on homepage
- Any additional service or product images

Recommended image sizes are detailed in the main README.md file.
